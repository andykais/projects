This project is based off of the Serpinski Triangle's Chaos Game
It says from a random starting point, if the midpoint between any three endpoints is chosen, and a new midpoint is continually chosen, from an endpoint and the midpoint, a dense fractal triangle will arise.
I took this method and applied it to polygons with more than 3 endpoints. The results can be quite beautiful.
Serpinksi's Triangle
Choose how many endpoints and use the mouse to place them on the canvas
